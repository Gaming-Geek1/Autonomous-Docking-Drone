# Autonomous-Docking-Drone
An autonomous hex copter that can lift two pounds for thirty minutes dock and can how swap batteries and payloads all autonomously. It will use Ardupilot and open source flight controllers.
This can be use for many things from cargo delivery, survalence, serch and rescue, videography and much more.
I made this project because I have a passion for developing automated systems and drones in general.
